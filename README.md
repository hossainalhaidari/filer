# Filer

Minimal file manager built with Go and TypeScript.

## Features

- [x] List Dirs
- [x] Read Files
- [x] Create Files/Dirs
- [x] Edit Files
- [x] Delete Files/Dirs
- [x] Rename Files/Dirs
- [x] Copy Files/Dirs
- [x] Move Files/Dirs
- [x] Change Files/Dirs Mode
- [x] Upload Files
- [x] Download Files
- [ ] Zip Files/Dirs
- [x] Unzip Files
